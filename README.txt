In this assignment I took already existing code and modified it so that it would count to 39, then reset to zero. There are four inputs, enable, clock, and reset, and an 8 bit wide terminal count. There are 2 outputs, an 8 bit wide count and a puple. Then the flipflop code is written, if reset goes high, the count reset to zero, otherwise, if enable is high, it checks to see if the couunt is equal to the terminal count, if it is, it resets to zero, otherwise it increases the count by 1. Then in the last line it assigns pulse to enable and makes count equal to terminal count, this way it will pulse once when enable is high and when it reaches the highest count, which is 39 in this case. Then in the test bench, clock, reset, and enable are created as registers, then ones_pulse is created as a wire, and ones count and tens count are created as a 8 bit wide wire. Initially, the clock is equal to zero and the clock is set up. Then, the previous module code is instantiated twice, one as a counter for the ones place, and one for the tens place. In the ones place counter, it counts up to nine and then rolls over, the tens place counts up to three, or thirty and then rolls over at 39. Then the reset signal and enable signals are tested in another initial block. The program successfully counts to thirty nine and then resets back to one.
